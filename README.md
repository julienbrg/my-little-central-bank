# My little central bank

[https://mylittlecentralbank.com/](https://mylittlecentralbank.com/)

Issue as euros just like a central bank would do.

## Install

```shell
npm i
```

Replace the `.env.example` with your own `.env` file.

## Test

```shell
npm start
```
